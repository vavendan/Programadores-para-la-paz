Sistema operativo:
Es el software base que administra los recursos del computador o servidor. Permite ejecutar aplicaciones y gestionar procesos.

Node.js:
Es un entorno de ejecución que permite ejecutar código JavaScript fuera del navegador. Se utiliza para crear servidores y aplicaciones web.

Aplicación server.js:
Es el archivo que contiene la lógica del servidor. Define las rutas y las respuestas que recibirán los usuarios.

Puerto:
Es un punto de comunicación que permite al servidor recibir solicitudes. En esta actividad se utiliza el puerto 3000.

Logs:
Son registros de eventos y actividades del sistema o la aplicación. Ayudan a monitorear el funcionamiento y detectar errores.
